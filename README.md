# Презентация: Физика

## Задание

Создать интерактивную презентацию на тему физики, рассмотреть единицы измерения с использованием современных веб-технологий.

## Требования к презентации

### Общие требования:
1. Презентация должна быть выполнена с использованием Markdown и HTML/CSS/JavaScript
2. Тема оформления вынесена в отдельный CSS файл
3. Минимальное количество слайдов: 20
4. Перед каждым разделом используется слайд-разделитель
5. Оформление должно быть читаемым (корректные цветовые контрасты)

### Обязательные элементы:
1. Титульный слайд
2. Заключительный слайд
3. Меню для быстрого перехода к слайдам
4. Навигация (стрелки, номера слайдов)

### Содержательные разделы:

#### 1. Единицы измерения (7 категорий, не менее 5 единиц в каждой):
- Скорость
- Температура
- Длина
- Масса
- Площадь
- Объем
- Давление

Для каждой категории необходимо показать:
- Как выводятся единицы
- Где применяются

#### 2. JavaScript конвертеры величин:
Создать конвертеры для 4 категорий из списка выше, например:
- Градусы Цельсия ↔ Фаренгейта ↔ Кельвины
- Километры ↔ Мили ↔ Футы
- Килограммы ↔ Фунты ↔ Унции
- и другие

#### 3. Задачи на скорость, время, расстояние:
- Теоретическое описание
- Примеры решения
- Формулы и объяснения

#### 4. Интерактивный калькулятор:
Программа на JavaScript для решения задач на скорость/время/расстояние.
На входе 3 input-поля. Для расчета третьего значения должны быть заполнены два других.

#### 5. Исходный код программ:
Отдельные JavaScript файлы с кодом конвертеров и калькулятора.
